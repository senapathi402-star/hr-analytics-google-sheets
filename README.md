## 📊 Project Overview
An automated HR Analytics Dashboard built with Google Sheets and Google Apps Script. The tool processes a 1000-row HR dataset and generates insights with one click — no manual work required!

## ✨ Features

### 1. 🟢 Generate Status
- Automatically classifies all 1000 employees as **"Active"** or **"Left Company"**
- Color codes each row — 🟢 Green for Active, 🔴 Red for Left
- Runs on entire dataset in seconds

### 2. 🔴 Highlight Attrition
- Scans all 1000 rows automatically
- Highlights entire rows of attrition employees in red
- Clears previous formatting before applying new

### 3. 📋 Generate HR Report
- Creates a fresh **"Reports"** sheet automatically
- Calculates and displays key HR metrics:

| Metric | Value |
|--------|-------|
| Total Employees | 1000 |
| Male Employees | 568 |
| Female Employees | 432 |
| Attrition Count | 201 |
| Attrition Rate | 20% |
| Average Salary | $1,813,251 |

## 🚀 How to Use
1. Open the Google Sheet
2. Click **HR Tools** in the menu bar
3. Choose any of the 3 options:
   - **Generate Status** → Classifies all employees
   - **Highlight Attrition** → Highlights employees who left
   - **Generate HR Report** → Creates summary report

## 🔍 Key Insights
- 📊 Marketing has highest headcount (112 employees)
- 💰 HR department has highest average salary ($1.95M)
- 📉 20% attrition rate — lower salary employees left more
- 👥 Male/Female split: 56.8% / 43.2%
